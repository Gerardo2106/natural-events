# HW3: Natural Events Tracker

## Description
A Node.js Express web app that tracks current natural events happening around the world and displays country information.

## Web APIs Used
- **NASA EONET (Earth Observatory Natural Event Tracker)** - Used to fetch and display current natural events such as wildfires, severe storms, volcanoes, and sea & ice events.
- **Pixabay API** - Used to display a random nature/disaster image on the home page.
